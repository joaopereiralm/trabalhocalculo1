# Calculadora de Frações - Cálculo 1

Aplicação desenvolvida em Python para a disciplina de Cálculo 1. 
O programa realiza operações de soma, subtração, multiplicação e divisão de frações 
utilizando a biblioteca nativa `fractions` para garantir a precisão matemática 
e simplificação automática dos resultados.

## Funcionalidades
- Operações com frações.
- Validação de entradas (impede o uso do número 0).
- Interface via terminal com menu interativo.
